Ultima2 Project plugins open-sourced

12/26/2020

As you maybe saw on the Other Projects Page, I (quite a long time ago) re-started to work on a Minecraft server from nothing with a friend of mine. But we quite got away from Minecraft and this project never came out.

This is why today I'm open-sourcing every plugin I worked on. And I hope it will help some people ! (The code is quite old so it's not the best plugins you can see but it does work really good (We did recreate Essentials plugin to not include it, it's always better when you do it).

Everything is here: UltimaWSLinker, UltimaUtil, UltimaGames and UltimaLobby
